Passion Works


Javascript   |   jQuery   |  CSS


I tend to consider Web development as a collection of puzzles. I like to pick alluring puzzles and try to solve them. As a junior web developer, first I define what exactly I want, then I plan the steps: given the input X, what are the steps needed to return output Y. Then I divide it into sub-parts and do some research to get some clues. Finally, I'm ready to develop my solution, debug it and... Done!
